Forgot Password

Use the "Forgot Password" option on the Sign In screen. We'll send a 6-digit reset code to your registered email address. The code expires in 15 minutes.

👶

Adding Children

Go to Parent Dashboard → Settings. Create a child profile with a nickname and avatar. Share the family code so they can join from their own device.

🗑️

Delete Account or Data

Parents can delete child profiles from Settings. To delete your entire account, email us at support@dodotx.net and we'll process it within 30 days.

Frequently Asked Questions

Is DodotX free?

DodotX is free to download and use with all core features. We may introduce optional premium features in the future.

Does my child need an email to use DodotX?

No. Children join using a parent-generated family code and a nickname. No email, password, or personal information is needed for children.

What is the family code?

The family code is a 6-character code generated by the parent. It expires every 60 minutes for security. You can regenerate it anytime from Settings.

Can grandparents view my child's progress?

Yes! Use the "Visitor View" feature. Share the family code with them — they can view progress without creating an account. It's completely read-only.

Is my child's data safe?

Absolutely. We're COPPA compliant, collect minimal data (nickname + age only), use no tracking or ads, and parents have full control over all child data. See our Privacy Policy for details.

What platforms does DodotX support?

DodotX is available on iOS, Android, and web. You can use it on phones, tablets, and desktop browsers.
